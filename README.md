### Motivation
When I use Are.na to collect ideas and other fragments, I want to organize ideas in relation to each other. This project is a baby step in trying to build tooling to support my personal wanderings on the web. It's serving as a playground, so lots of sand and scraps! ğŸ›ğŸŒ

 
### Features
* Import blocks from any public channel using the channel URL
* Reposition and expand/minimize blocks
* View embedded videos and websites in context
* Smooth zooming and panning (custom implementation)

### Built with
* ReactJS
* [React-draggable](https://www.npmjs.com/package/react-draggable)
* [Are.na API V2](https://dev.are.na/documentation/channels)
  
### Installation
Run locally with
```
npm install
npm run dev
```
